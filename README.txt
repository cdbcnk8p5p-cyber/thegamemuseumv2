THE GAME MUSEUM — VERSION 3.5

NEW IN 3.5
- EA Sports FC 24 and EA Sports FC 25 now use the supplied full Xbox Series X case images.
- The physical case and top platform banner remain visible in catalogue cards and the large catalogue popup.
- The large centred game-detail layout from v3.4 is retained and polished as the museum standard.
- Mafia: The Old Country and Grand Theft Auto V are unchanged for now.
- Version numbers, browser storage migration, export name, manifest and offline cache are updated to v3.5.
- Existing v3.4 browser data is migrated automatically.

UPLOAD TO GITHUB PAGES
1. Extract this ZIP.
2. Open the the-game-museum-v3.5 folder.
3. Upload every file and the assets folder to the root of the existing GitHub repository.
4. Replace the old files when prompted.
5. Wait a minute, then refresh the live site.
6. If an older cached version appears, refresh once more or close and reopen the installed app.
