THE GAME MUSEUM V2 — GITHUB PAGES UPLOAD

1. Extract this ZIP.
2. Open the folder named the-game-museum-v2.
3. Upload every file INSIDE that folder to the root of your GitHub repository.
4. Make sure index.html is visible on the main repository page.
5. Settings > Pages > Deploy from a branch > main > /(root) > Save.
6. Wait for deployment, then open the Pages link.

The spreadsheet is not changed. App additions are stored locally; use Export app backup after important changes.
